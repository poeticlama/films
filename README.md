# Films

Небольшое приложение для поиска фильмов, просмотра карточки фильма, добавления в избранное и сравнения.

## Как запустить

Требуется Node.js 20+.

1. Установить зависимости:

```bash
npm install
```

2. Создать файл `.env` в корне проекта и добавить ключ API (получить можно в [тг-боте](https://t.me/poiskkinodev_bot)):

```env
VITE_API_KEY=your_api_key
```

3. Запустить проект в режиме разработки:

```bash
npm run dev
```

4. Сборка production-версии:

```bash
npm run build
```

5. Локальный просмотр production-сборки:

```bash
npm run preview
```

## Что реализовано

### Хуки

- `useFilms` в `src/hooks/api.ts` - получение списка фильмов с фильтрами и подгрузкой по скроллу. Использован запрос "Универсальный
   поиск с фильтрами" (https://api.poiskkino.dev/v1.5/movie)
- `useFilm` в `src/hooks/api.ts` - загрузка детальной информации о фильме по `id`. Использован запрос "Поиск по id" (https://api.poiskkino.dev/v1.4/movie/{id})

### Компоненты и страницы

- `FilmCard` - карточка фильма (постер, название, год, рейтинг).
- `GenreFilter` - фильтр по жанрам (мультивыбор).
- `FilmsPage` - основная страница со списком фильмов, поиску по фильтрам.
- `FilmPage` - детальная страница фильма, добавление/удаление из избранного и сравнения.
- `FavoriteFilmsPage` - список избранных фильмов (для их хранения используется `localStorage`).
- `ComparisonPage` - таблица сравнения фильмов (для их хранения используется `localStorage`).

## Используемые библиотеки (кроме React)

- `react-router` - Declarative Mode, маршрутизация между страницами.
- `@siberiacancode/reactuse` - `IntersectionObserver` для бесконечного скролла.
- `tailwindcss` и `@tailwindcss/vite` - стилизация интерфейса.
- `vite` - dev-сервер и сборка.
